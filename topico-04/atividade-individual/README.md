# Atividade Individual - Tópico 04

## Objetivo

Esta atividade tem como objetivo aplicar medidas iniciais de segurança a um serviço web publicado em sistema Linux, identificando portas, serviços ativos e aplicando regras básicas de firewall.

---

## Conteúdo do projeto

- superficie-ataque.md → análise de portas e serviços ativos
- firewall.md → configuração do firewall UFW
- hardening.md → sugestões de melhoria de segurança
- validacao.md → validação do funcionamento do serviço
- comandos.txt → comandos utilizados na análise
- evidencias/ → capturas de ecrã dos testes realizados

---

## Serviço analisado

- Serviço web ativo na porta 80/TCP (HTTP)

---

## Ferramentas utilizadas

- Linux Mint
- UFW (firewall)
- systemctl
- ss (análise de portas)

---

## Resultado

Foi identificada a superfície de ataque do sistema, configurado o firewall com regras básicas e validado que o serviço web continua funcional após as alterações.
